# Bíblia Hinário CCB

Aplicativo móvel não oficial de Bíblia e Hinário da CCB desenvolvido com React Native e Expo.

## Recursos

- Aba de bíblia
- Aba de hinário
- Pesquisa de livros e hinos
- Modo claro ou escuro que acompanha a configuração do Android
- Sem propagandas e permissões
